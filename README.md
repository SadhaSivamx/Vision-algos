### Vision-algos
A collection of computer vision algorithms and experiments.
<br>
### CodeSections 
```text
vision-algos
│
├── /ObjectTracking
│   ├── /Src
│   │   └── Drone.mp4 ( Video )
│   │   └── vid2img.py ( Turns Video to Frames )
│   └── ByTemplateMatching.py ( Tracking Done with TM )
│   └── ByFeatureMatching.py ( Tracking with Sift Features )
│   └── Inloop.py ( Combined )
│   └── Yolomodel.py ( Code for Getting ROI from Model )
│   └── Best.onnx ( DL Model )
└── README.md
```
